## 1. **Nama**
Layanan visa
## 2. **Deskripsi**
Website ini adalah layanan visa satu pintu di Indonesia yang menawarkan kemudahan pengajuan visa, perpanjangan visa, 
dan visa on arrival dengan tampilan modern dan fitur yang mudah digunakan.

## 3. Teknologi yang Digunakan
- HTML5
- CSS3 (Tailwind CSS)
- JavaScript
  
## 4. Cara Menjalankan Proyek Secara Lokal
- Clone repositori ini ke komputer lokal Anda:
 git clone https://github.com/Jois01/visa.github.io.git
- Masuk ke direktori proyek:
 cd nama-repositori
- Buka file melalui text editor seperti VSCode, latu buka terminal dan jalankan :
 npx tailwindcss -i ./src/input.css -o ./src/output.css --watch
- Lalu klik kanan pada file index.html dan pilih "Open in Browser."

Jelaskan bagaimana proses pengembangan proyek, dan tangtangan yang di hadapi:

```markdown
## Perkembangan Proyek
- Pembuatan kerangka website
- Pembuatan mendesain website dari, warna, ukuran font, jenis font, penambahan icon dan penambahan gambar.
- Membuat layout responsif.
- Pengujian website di berbagai perangkat untuk memastikan tampilannya responsif.

## Tantangan yang Dihadapi
1. Masalah Layout Responsif : elemen-elemen seperti gambar dan teks tidak tampil dengan baik di berbagai ukuran layar.
  Solusi : Saya menggunakan grid dan flexbox.
2. Overflow pada Halaman : Halaman website memiliki ruang kosong di sisi kanan.
Solusi : Mengecek apakah ada padding berlebih yang saya berikan, dan langsung memperbaikinya.
